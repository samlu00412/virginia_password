# 維吉尼亞密碼
本程式透過C++實現維吉尼亞密碼的加密與解密操作，以達成日常生活傳遞敏感訊息的保密性
## 安裝說明
本程式無須安裝，只需編譯main.cpp即可產生執行檔
## 使用方法
1. 開啟main.cpp後，即可使用IDE(例如VScode、Dev-C++、Codeblock等)編譯程式，並產生執行檔
2. 開啟執行檔，待程式輸出說明後進行操作
